熟练运用Spring(番外篇)：自定义注解+JdbcTemplate实现无sql的crud

1、注解

  1. @interface 注解关键字
  2. String value(); 注解的属性，可支持默认值设置（String value() default “”;）
  3. @Target 作用域
    • ElementType.TYPE 类或接口
    • ElementType.FIELD 字段
    • ElementType.METHOD 方法
    • ElementType.PARAMETER 参数
    • ElementType.CONSTRUCTOR 构造器
      等等。
  4. @Retention 分类
    • RetentionPolicy.RUNTIME 运行时存在，可通过反射读取
    • RetentionPolicy.CLASS 编译时会记录在class文件中，运行时忽略
    • RetentionPolicy.SOURCE 只在源码显示，编译时会丢弃
  5. @Inherited 允许子类继承
  6. @Documented 生成javadoc时会包含注解
